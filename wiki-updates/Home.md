Welcome to the 1D_SPH wiki!

1D_SPH is a one-dimensional smoothed particle hydrodynamics (SPH) simulation code written in Julia.

There are some options for the following components:
* Formulation of SPH equations
* Kernel interpolation
* Kernel function
* Volume element
* Time integrator
* Time-dependent viscosity

1D_SPH はJulia言語で書かれた1次元SPHシミュレーションコードです。

1D_SPHでは、以下の要素についていくつかの選択肢からユーザーがスキームを選択するすることができます:

* SPH方程式系の定式化
* カーネル補間法
* カーネル関数
* 体積要素
* 時間積分法
* 時間依存人工粘性
